## 🤝 Code Convention

> 1. 변수 네이밍 규칙 : ```Camel Case```
> 2. 한 줄의 글자 수 : ```최대 80자```
> 3. 한 줄 주석 표기 : ```/*  */```
> 4. 여러 줄 주석 표기 : 
```java
  /*-
   *
   *
   */
```
> 5. 삼항연산자 표기 : 
```
  alpha = (BooleanExpression) ? beta
                              : gamma
```
> 6. If문에서 &&, ||' 표기
```
  if((condition1 && condition2)
    && (condition3 && condition4) 
    && (condition5 && condition6) {
    do something();
  }
```
